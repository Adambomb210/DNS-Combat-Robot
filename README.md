# DNS-Combat-Robot
WIP
A beetleweight forked undercutter.
<img width="1113" height="699" alt="Screenshot 2026-03-22 211701" src="https://github.com/user-attachments/assets/b30b0a7f-dfa5-4d0d-9fce-2e334620e239" />

This is a 4 wheel drive beast, using the JCR Cherry Bomb motor and the Blitz Lites for drive.

The CAD can be viewed here: https://cad.onshape.com/documents/54b35853b490f7b636402c15/w/556437aa5a32b67c4960c036/e/38ae217d228fbcbd10d8c5af?renderMode=0&uiState=69d9bd39969e9bf0c96ec368

I made this project as I have been going to NHRL for years as a spectator, and for all those years I told myself I would be behind the sticks one day. Well, that day is coming very soon, hopefully April 2026, but maybe May. On top of that, I just like engeneering and I think this has been be a great learning opritunity.
